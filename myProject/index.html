Layout de menu lateral

Para usar este layout COPIE o HTML e o CSS side-menu.css, e com JavaScript em index-ui.js. O JS usa o VANILLA para marcar um item como ativado o que faz com que o menu seja responsivo.

Imagens possuem direitos autorais

Como eu simplesmente só copiei e editei alguns elementos, as imagens permaneceram. Elas podem conter direitos autorais, então, cuidado. Fiz isso apenas para testar alguns elementos. Fiz uma conexão simples entre as páginas de HOME e PREÇOS.

Entre em modo desenvolvedor!

Entre no modo desenvolvedor do seu navegador, eu estou usando o Firefox, então precionei os comandos (CTRL+SHIFT+M) para entrar em modo de visualização móvel (smartphones).
